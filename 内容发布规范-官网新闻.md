## 一、关键词选择规则

### 1.1 关键词构成
```
1个一级词（核心业务） + 2个二级词（主营产品） + 3个长尾词（相关话题）
```

### 1.2 关键词分类

**一级词（核心业务）**：
| 关键词 |
|--------|
| 一人公司 |
| 一人部门 |
| AI Agent |
| 智能体 |

**二级词（主营产品）**：
| 关键词 |
|--------|
| 网站建设 |
| 网站推广 |
| 网站制作 |
| 网站优化 |
| SEO |
| 小程序开发 |
| 软件开发 |
| 系统开发 |
| 软件公司 |
| 网络公司 |

**三级词/长尾词**（相关话题）：
| 关键词 |
|--------|
| 数字化转型 |
| 降本增效 |
| 企业服务 |
| 创业支持 |
| 智能办公 |
| 高效运营 |

### 1.3 选词规则
1. 从一级词表中选1个
2. 从二级词表中选2个（不重复）
3. 从长尾词表中选3个（与内容相关）
4. 确保同一城市的一级词+二级词组合有业务关联性

**示例**：
- 一级：一人公司
- 二级：网站建设、小程序开发
- 长尾：数字化转型、创业支持、高效运营

## 二、内容模板

### 2.1 新闻标题
```
[城市][一级词核心词]与[二级词1][二级词2]如何[长尾词/动词短语]
```

**示例**：
- 盘锦一人公司与网站建设小程序如何助力高效创业
- 沈阳AI Agent与软件开发如何赋能企业数字化转型
- 大连智能体与网站推广如何实现降本增效

### 2.2 新闻关键词（meta keywords）
```
[城市][一级词],[城市][二级词1],[城市][二级词2],[长尾词1],[长尾词2],[长尾词3]
```

**示例**：
```
盘锦一人公司,盘锦网站建设,盘锦小程序开发,数字化转型,创业支持,高效运营
```

### 2.3 新闻简介
- 50-150字
- 包含核心关键词
- 说明文章解决什么问题

### 2.4 正文结构（4段）
```
第一段：引入话题（痛点/现象）
第二段：展开分析（问题/方案）
第三段：案例/数据支撑
第四段：千羽网络认为（结合主营产品提解决方案）
```

### 2.5 第四段写作公式
```
千羽网络认为：[核心观点]。

[具体解释AI Agent与企业工具结合的价值]。

千羽网络的[相关产品/服务]，正是将AI Agent与[具体系统]深度集成，让AI真正成为企业的数字化员工。
```

**示例第四段**：
```
千羽网络认为：AI Agent提升办公效率多少，与给它配备什么样的工具密切相关。

AI Agent的核心价值在于数据处理和决策辅助。如果配合专业的业务系统，如AI智能CRM、AI智能销售助手等，可以让AI直接操作企业数据，自动生成客户跟进记录，分析销售机会，生成报表等。

千羽网络提供的一人部门解决方案，正是将AI Agent与CRM、ERP等专业系统深度集成，让AI真正成为企业的数字化员工。
```

## 四、发布流程

### 4.1 准备阶段
1. 确定发布日期
2. 从关键词表选择关键词组合
3. 确定文章主题和角度

### 4.2 创作阶段
1. 根据模板生成标题
2. 生成meta keywords
3. 撰写简介（50-150字）
4. 撰写正文4段（700字+）
5. 特别注意第四段要结合千羽网络产品

### 4.3 文件创建
1. 创建详情页HTML文件（基于article-template.html）
2. 替换所有占位符
3. 添加标签
4. 添加相关推荐

### 4.4 索引更新
1. 编辑 news/index.html
2. 在对应Tab（公司新闻/行业资讯）顶部插入新闻条目
3. 链接到详情页

### 4.5 发布
1. Git提交
2. Git推送
3. 执行百度主动推送（如配置）

## 八、注意事项

1. **第四段必须结合产品**：这是与竞品差异化的关键
2. **标题要有吸引力**：使用问句或数字增加点击欲
3. **内容要专业可信**：避免绝对化用语（最、第一、绝对）
4. **关键词自然嵌入**：不要堆砌关键词
5. **发布后记录**：及时更新关键词使用记录表
